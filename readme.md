此插件原用于吉他谱的下载和拼接

功能：下载当前页面的图片然后拼接在一起。

### 使用方式

1. 进入 `chrome://extensions/`，打开「开发者模式」，点击「加载已解压的扩展程序」，选择本目录。
2. 在任意页面点击扩展图标（或弹窗里的「打开悬浮窗」按钮），右下角会出现悬浮卡片。
3. 悬浮卡片会列出当前页所有图片（预览、宽高、大小），可按大小过滤（默认≥200KB）并切换横/纵向拼接，多选后执行「下载所选」或「拼接所选」，完成后可点右上角「×」关闭。
4. 拼接默认横向（过宽会自动改为纵向），下载与拼接保存到 `pindekuai/<域名>/`（优先沿用原图文件名，找不到时以 `pindekuai-序号` 命名），且不再弹出保存对话框。

如遇 “Receiving end does not exist” 的提示，请确认当前标签页是音伴站点；插件会自动通过后台脚本注入后重试。如果仍失败，请刷新页面或升级浏览器版本再试。
